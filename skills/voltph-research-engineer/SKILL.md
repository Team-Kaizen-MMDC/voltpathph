# Voltpath PH Research Engineer Skill

Expert guidance for investigative tasks, prototyping, and technical research within the Voltpath PH ecosystem.

## 🔬 Research Domains
- **Predictive Modeling:** Investigating traffic patterns and environmental factors (weather, elevation) affecting EV range in the PH.
- **Experimental APIs:** Evaluating 3rd party data sources (e.g., PH government data, specialized weather APIs) for integration.
- **Algorithm Optimization:** Refining the "math" behind battery drain predictions and charging stop recommendations.
- **Market Trends:** Keeping the EV catalog up to date with new models entering the Philippines (e.g., Vinfast, Omoda).

## 🚀 Key Workflows

### 1. Technical Feasibility Studies
- When a new feature is proposed (e.g., "Solar charging prediction"), perform a deep dive into available data and libraries.
- Create small "spike" prototypes in the `packages/shared` or as standalone scripts to validate assumptions.

### 2. API Evaluation
- Compare different providers for traffic and map data (e.g., Google vs. Here vs. Mapbox) specifically for coverage accuracy in PH provincial areas.
- Document costs, rate limits, and data freshness.

### 3. Prototyping
- Use the `shared` package logic to build experimental calculation models.
- Validate new algorithms against historical trip data if available.
