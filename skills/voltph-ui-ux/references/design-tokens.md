# Voltpath PH Design Tokens

Core visual variables for a consistent and premium EV charging experience.

## 🎨 Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| **Electric Green** | `#22C55E` | Primary actions, "Charged" status, EV branding |
| **Charge Blue** | `#3B82F6` | Secondary actions, Info, Progress |
| **Carbon Black** | `#0F172A` | Backgrounds (Dark Mode), Headings |
| **Cloud White** | `#F8FAFC` | Backgrounds (Light Mode), High contrast text |
| **Asphalt Gray** | `#64748B` | Subheadings, Borders, Disabled states |
| **Warning Amber** | `#F59E0B` | Low battery, Offline station, Alerts |

## Typography
- **Primary Font:** Inter, system-sans-serif
- **Headings:** Semi-bold/Bold, Carbon Black
- **Body:** Regular, Slate 600
- **Monospace:** JetBrains Mono (for coordinates/ID values)

## Spacing & Layout
- **Grid:** 8px base unit
- **Border Radius:** 12px (Standard), 8px (Small), Full (Pills)
- **Shadows:** Soft, multi-layered (e.g., `0 4px 6px -1px rgb(0 0 0 / 0.1)`)

## Interactive States
- **Hover:** Slight scale up (1.02) + brightness increase
- **Active/Pressed:** Slight scale down (0.98)
- **Focus:** 2px ring in Electric Green
