# Voltpath PH Documentation Engineer Skill

Maintaining the clarity and accuracy of the Voltpath PH knowledge base.

## 📝 Documentation Standards
- Use **GitHub Flavored Markdown**.
- Include Mermaid diagrams for architecture and process flows.
- Use version and tech stack badges in all primary READMEs.

## 🚀 Key Workflows

### 1. Updating Diagrams
- Use Mermaid syntax in Markdown files.
- Ensure `docs/ARCHITECTURE.md` is updated whenever the system design changes.

### 2. Onboarding Maintenance
- Regularly review `docs/ONBOARDING.md` to ensure setup steps are accurate for new hires.
